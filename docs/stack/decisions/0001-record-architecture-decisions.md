# ADR 0001: Record architecture decisions

## Status

Accepted

## Context

The project is documentation-first. Stack choices will accumulate; without a record, rationale is lost and agents or new contributors may re-litigate settled questions.

## Decision

Use **Architecture Decision Records** in this directory: one topic per file, numbered prefix, status field, and links to superseding ADRs when replaced.

## Consequences

- New technical choices should add a new `NNNN-title.md` or update status of an existing one.
- The template lives in [0000-adr-template.md](0000-adr-template.md).
